# A simple terminal text editor written in Rust using the termion library


Warning: a lot of bugs, development halted due to personal reasons

# Usage

1) `cargo r` to compile and run the project
2) Editor starts off in "INSERT MODE" (same as vim)
3) To quit, press ESC and then write :q (no other commands supported as of now)
